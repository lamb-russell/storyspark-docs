# Story Spark documentation

Public documentation site for **Story Spark**, an iPhone app that tells voice-first
interactive stories for kids aged 4–7.

The site is a single static page ([index.html](index.html)) served with GitHub Pages.

- How the app works and how to set it up
- Getting and using a free Gemini API key
- Using the on-device Apple Intelligence story model
- Troubleshooting and privacy details
